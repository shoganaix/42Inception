# Inception
**The objective of this project is to create a set of Docker containers to simulate a multi-service environment, similar to a mini production environment, deploying services like Nginx, MySQL, WordPress, and custom applications.**

> [!NOTE]
> This project teaches containerization, networking, and service orchestration using Docker and Docker Compose.

## Project Description
Inception is a project where you build a **Docker-based environment** from scratch:

- Using Dockerfiles to build custom images
- Setting up Docker Compose to orchestrate containers
- Configuring networking between containers
- Deploying services such as Nginx, MySQL, WordPress
- Exposing services to the host machine

---

## Services
- **Nginx** ✔️
- **MariaDB** ✔️
- **WordPress** ✔️
- **Adminer** ❌
- **FTPS / custom app containers** ❌
- **NO bonus services** ❌

## Usage

1. Clone this repository:

```bash
git clone https://github.com/shoganaix/42Inception
cd Inception
```

2. Compile:

```bash
make
```
